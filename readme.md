# ollama-commit

# before all

1. install [ollama](https://ollama.com/).
2. pull [llama3.1](https://ollama.com/library/llama3.1)

```bash
ollama run llama3.1:latest
```

# feature

- use `ollama` to generate commit message
  
# install

```bash
npm i -g @miaos/ollama-commit
```

# usage

```bash
ollama-commit
```
